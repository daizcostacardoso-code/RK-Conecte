# Tipos de Servico

Este documento lista os principais tipos de servico que devem orientar o fluxo
comercial e o futuro orcamento inteligente do RK-Conecte.

## Box

Servico para banheiro, geralmente com vidro temperado, ferragens e instalacao.

Campos relevantes:

- Tipo de abertura.
- Largura.
- Altura.
- Quantidade de folhas.
- Tipo de vidro.
- Cor das ferragens.
- Observacoes de parede, piso e prumo.

## Espelho

Servico de fornecimento e instalacao de espelho.

Campos relevantes:

- Largura.
- Altura.
- Quantidade.
- Lapidacao.
- Bisote, quando houver.
- Instalacao com cola, parafuso ou suporte.
- Recortes ou tomadas.

## Guarda-corpo

Protecao em vidro para escadas, sacadas, varandas e mezaninos.

Campos relevantes:

- Comprimento total.
- Altura.
- Tipo de fixacao.
- Tipo e espessura do vidro.
- Corrimao, quando houver.
- Local de instalacao.
- Necessidade de visita tecnica.

## Fachada

Fechamento ou composicao frontal para loja, comercio ou residencia.

Campos relevantes:

- Largura total.
- Altura total.
- Tipo de vidro.
- Perfis de aluminio.
- Portas integradas.
- Travessas ou divisorias.
- Condicoes de acesso ao local.

## Cobertura

Cobertura em vidro, policarbonato ou solucao combinada.

Campos relevantes:

- Area.
- Tipo de estrutura.
- Inclinacao.
- Tipo de vidro ou material.
- Vedacao.
- Calhas ou escoamento.
- Risco e acesso para instalacao.

## Porta

Porta de vidro, aluminio ou sistema combinado.

Campos relevantes:

- Largura.
- Altura.
- Tipo de abertura.
- Vidro.
- Ferragens.
- Puxador.
- Mola ou fechadura.

## Janela

Janela de vidro, aluminio ou sistema sob medida.

Campos relevantes:

- Largura.
- Altura.
- Tipo de abertura.
- Quantidade de folhas.
- Vidro.
- Perfil.
- Tela ou acessorio.

## Projeto personalizado

Servico fora dos modelos comuns.

Exemplos:

- Divisoria especial.
- Mesa de vidro.
- Prateleira.
- Fechamento de area.
- Solucao sob medida para obra.

Campos relevantes:

- Descricao livre.
- Fotos.
- Medidas.
- Materiais desejados.
- Observacoes tecnicas.
- Necessidade de visita.

## Diretriz para UX

O usuario deve primeiro escolher o tipo de servico. Depois, o sistema deve
mostrar campos adequados para aquele servico, evitando formularios longos e
genericos demais.
