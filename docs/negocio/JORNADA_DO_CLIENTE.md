# Jornada do Cliente

A jornada do cliente descreve como uma oportunidade passa pelo RK-Conecte desde
o primeiro contato ate o pos-venda.

## Fluxo macro

```text
Lead -> Cliente -> Projeto -> Orcamento -> PDF -> Aprovacao
-> Producao -> Instalacao -> Financeiro -> Pos-venda
```

## 1. Lead

Pessoa ou empresa que demonstrou interesse, mas ainda nao possui cadastro
completo ou Projeto ativo.

Entradas comuns:

- WhatsApp.
- Indicacao.
- Ligacao.
- Atendimento presencial.
- Formulario do site.
- Cliente antigo solicitando novo servico.

Informacoes minimas:

- Nome.
- Telefone.
- Interesse inicial.
- Cidade ou bairro.
- Observacao do pedido.

## 2. Cliente

Lead com dados suficientes para ser acompanhado pela vidracaria.

Informacoes esperadas:

- Nome.
- Telefone principal.
- Telefone alternativo, quando houver.
- Email, quando houver.
- Documento, quando necessario.
- Endereco principal.
- Observacoes de relacionamento.

## 3. Projeto

Unidade central de trabalho. Representa a demanda real do cliente e acompanha o
fluxo comercial, operacional e financeiro.

Um cliente pode ter varios Projetos.

Exemplos:

- Box para banheiro social.
- Fachada comercial.
- Guarda-corpo de varanda.
- Espelhos para apartamento.

## 4. Orcamento

Estimativa tecnica e comercial vinculada ao Projeto.

O orcamento deve conter:

- Tipo de servico.
- Medidas.
- Itens.
- Materiais.
- Servicos adicionais.
- Descontos e acrescimos.
- Forma de pagamento.
- Validade da proposta.

## 5. PDF

Documento comercial enviado ao cliente.

Objetivo:

- Apresentar proposta clara.
- Formalizar escopo, valores e validade.
- Facilitar aprovacao.
- Evitar duvidas sobre o que esta incluso.

## 6. Aprovacao

Momento em que o cliente aceita a proposta.

Sinais de aprovacao:

- Confirmacao por mensagem.
- Assinatura ou aceite formal.
- Pagamento de sinal.
- Autorizacao para iniciar producao.

Resultado esperado:

- Projeto passa para status aprovado ou ativo.
- Financeiro registra condicao combinada.
- Operacional recebe informacoes para producao.

## 7. Producao

Etapa de preparacao dos materiais, corte, compra, separacao ou organizacao para
execucao.

Informacoes importantes:

- Medidas finais.
- Tipo de vidro.
- Ferragens.
- Prazo.
- Responsavel.
- Observacoes tecnicas.

## 8. Instalacao

Execucao na obra do cliente.

Informacoes importantes:

- Endereco da obra.
- Contato no local.
- Data prevista.
- Equipe ou instalador.
- Restricoes de acesso.
- Fotos e observacoes.

## 9. Financeiro

Acompanhamento de valores e recebimentos.

Informacoes importantes:

- Valor aprovado.
- Valor recebido.
- Saldo.
- Forma de pagamento.
- Vencimentos.
- Pendencias.

## 10. Pos-venda

Etapa posterior a conclusao.

Objetivos:

- Confirmar satisfacao.
- Registrar garantia ou ajustes.
- Manter historico do cliente.
- Facilitar novas vendas.

## Pontos de cuidado

- O cliente nao deve precisar repetir informacoes.
- Cada Projeto deve guardar seu historico.
- O PDF deve refletir o orcamento aprovado.
- A aprovacao deve ser clara para evitar producao indevida.
- Financeiro deve receber informacoes no momento certo.
