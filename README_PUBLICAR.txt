RK Vidraçaria - Pacote funcional

Como publicar:
1. Extraia este ZIP.
2. Abra o PowerShell dentro da pasta site_firebase_online.
3. Execute: firebase.cmd deploy
4. Abra o site e pressione Ctrl + F5 para limpar cache.

Estrutura correta:
site_firebase_online/
  firebase.json
  .firebaserc
  site_lancamento/
    index.html
    css/
    js/
    imagens/
    paginas/
